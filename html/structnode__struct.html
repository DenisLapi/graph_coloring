Detailed Description

This is the structure of nodes that we are creating for Welsh-Powell algorithms We are using this structure to compare the properties between nodes such as id, sum_connection, colors When we are resolving the graph coloring problem with Welsh-Powell algorithm, we are coloring nodes starting from the nodes with bigger number of connections.
